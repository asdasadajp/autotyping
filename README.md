# Zone Typing Hack

`zone-typing.pages.dev` 向けの Chrome 拡張機能です。入力されたキーを、画面に表示されている次の文字へ置き換えることで、タイピング判定を補助します。

> [!CAUTION]
> この拡張機能は学習・検証目的で使用してください。対象サービスの利用規約やルールを確認し、ランキング、試験、競技など不正行為につながる場面では使用しないでください。

## 機能

- 拡張機能のポップアップから補助機能の有効・無効を切り替え
- `Ctrl` + `Shift` で素早く切り替え
- 設定を Chrome のローカルストレージへ保存
- `zone-typing.pages.dev` のみに限定して動作

## インストール

1. このリポジトリをダウンロード、またはクローンします。

   ```bash
   git clone https://github.com/asdasadajp/autotyping.git
   ```

2. Chrome で `chrome://extensions/` を開きます。
3. 右上の「デベロッパー モード」を有効にします。
4. 「パッケージ化されていない拡張機能を読み込む」を選択します。
5. このリポジトリのフォルダを指定します。

## 使い方

1. [Zone Typing](https://zone-typing.pages.dev/) を開きます。
2. ツールバーの拡張機能アイコンを押します。
3. トグルスイッチで機能を有効にします。
4. 通常どおりキーを入力します。

`Ctrl` + `Shift` を押すことでも、有効・無効を切り替えられます。

## ファイル構成

| ファイル | 役割 |
| --- | --- |
| `manifest.json` | Chrome 拡張機能の設定と権限 |
| `popup.html` | ポップアップ画面 |
| `popup.js` | 設定の保存と切り替え処理 |
| `content.js` | 対象ページと拡張機能の橋渡し |
| `inject.js` | ページ内のキーボードイベント処理 |

## 権限

この拡張機能は、次の Chrome 権限を使用します。

- `activeTab`: 現在開いているタブへのメッセージ送信
- `scripting`: 対象ページ上でのスクリプト実行
- `storage`: 有効・無効の設定保存
- `https://zone-typing.pages.dev/*`: 対象サイトでのみ動作するためのホスト権限

## 開発

ビルド処理や外部パッケージは不要です。ファイルを変更した後、`chrome://extensions/` で拡張機能を再読み込みしてください。
